다번역 앱 설명서

1. 개요

무료로 제한 없이 레이아웃을 유지하면서 PDF 번역을 해보세요. 최고의 품질을 제공하는 PDF 번역기입니다. 한두 번의 탭으로 PDF 번역을 빠른 시간 내에 제한 없이 해줍니다. 무료인 경우 21개 언어, 유료인 경우 190개 언어로 쉽게 번역할 수 있습니다.

2. 주요 기능

  • 현재는 PDF 하나만 가능합니다.
  • 최대 190개 언어로 번역할 수 있습니다. (무료: 21개)
  • 언어를 감지하는 기능이 있습니다.
  • 4가지 방법으로 번역이 가능합니다.
    • 전체 번역
    • 글 선택
    • 펜 브러쉬
    • 글상자
  • 채팅 번역창을 제공합니다. (펜 브러쉬, 글상자)
  • 유료인 경우, 이미지 PDF 번역이 가능합니다. (무료는 불가)
  • 무료이면서 무제한으로 레이아웃을 유지하며 번역합니다.

3. 세부 기능 설명

앱 상단 툴바에 있는 왼쪽과 오른쪽에 있는 버튼들에 대해 설명합니다.

4. 기본 상태

4.1. 기본 모드

버튼을 선택하지 않고 PDF 파일을 열어놓은 상태에서는 글선택(TextSelection) 모드로 되어 있습니다.

4.2. 스크롤 및 선택

한 손가락으로 스크롤이 가능하며, 선택하려는 글을 길게 누르면 선택이 되면서 메뉴 버튼을 통해 기본 메뉴를 볼 수 있으며 번역 메뉴를 선택할 수 있습니다.

5. 왼쪽 버튼 그룹

  • 통합메뉴: 앱에 필요한 대부분의 기능을 위한 메뉴를 보여줍니다.
  • 파일열기: PDF 파일을 엽니다.
  • 파일저장: 변경된 소스 및 대상(번역된) 파일을 저장합니다.
  • 파일닫기: 열려있는 파일을 닫습니다.
  • 실행취소: 그리기(Drawing), 글상자(TextBox) 모드 등에서 화면에 그려진 것을 취소합니다.
  • 재실행: 실행취소한 것을 재실행합니다.

6. 오른쪽 버튼 그룹

  • 공유: 번역된 파일을 다른 사용자 또는 다른 앱과 공유합니다.
  • 그리기(펜) 모드: PDF 화면에 그리기를 할 수 있습니다.
    • 그리기 모드가 설정되면, 오른쪽 사이드에 있는 채팅창을 통해 그린 부분의 번역 내용을 확인할 수 있습니다.
    • 글자가 작거나 펜 글씨인 경우 인식하지 못할 수 있습니다.
    • 스크롤은 두 손가락으로 가능합니다.
  • 글 상자(TextBox):
    • 번역 시 매끄럽지 못한 글도 모아서 번역합니다.
    • 아이콘 설명과 키워드가 글상자로 선택되어 번역이 부드럽게 표현됩니다.
    • 현재 이 기능은 무료에서만 가능합니다. 유료는 추후 업데이트 예정입니다.
    • 스크롤은 두 손가락으로 가능합니다.
  • 글블럭(TextBlock) 보이기/숨기기:
    • 번역의 단위가 되는 글블럭의 구조를 보이거나 숨깁니다.
    • 번역이 매끄럽지 않다면 글상자를 사용해 끊어진 부분을 그룹핑할 수 있습니다.
  • 모든 주석 제거:
    • 그리기, 글블럭, 글상자 등의 PDF 주석을 모두 제거할 수 있습니다.
  • 계정 & 설정:
    • 유료 사용자를 위한 로그인 및 다양한 설정값을 보여주고 사용자가 변경할 수 있습니다.

7. 두번째 줄 버튼 그룹

  • 왼쪽 사이드바: 열려진 소스 PDF의 썸네일을 보여줍니다. 페이지를 이동할 수 있습니다.
  • 소스 PDF 정보: 소스 PDF의 파일 관련 정보를 볼 수 있습니다. (페이지, 텍스트블럭 수, 단어 수, 글자 수 등)
  • 소스 언어 선택: 소스 PDF의 언어를 선택할 수 있습니다. 언어 인식이 가능합니다. (무료: 21개, 유료: 190개)
  • 번역목록 버튼: 번역된 PDF 목록을 보여주며, 삭제하거나 다른 언어로 번역할 수 있습니다. (무료: 21개, 유료: 190개)
  • 대상 언어 선택: 대상 PDF의 언어를 선택할 수 있습니다. (무료: 21개, 유료: 190개)
  • 대상 PDF 정보: 대상 PDF의 파일 관련 정보를 볼 수 있습니다. (페이지, 텍스트블럭 수, 단어 수, 글자 수 등)
  • 번역채팅창:
    • 실시간으로 번역을 해주는 채팅창입니다.
    • 그리기 모드나 글상자 모드를 선택하여 번역 내용을 자동으로 보여줍니다.
    • 채팅 메시지도 번역할 수 있습니다.

8. 전체 기능 잠금해제 (유료 전환)

  • 왼쪽 메뉴 버튼을 누르면 "전체기능 잠금해제" 버튼이 있습니다.
  • 오른쪽 설정 버튼을 누르면 "로그인", "유/무료 선택", "광고/워터마크 제거" 버튼을 사용할 수 있습니다.
  • 언어 목록/언어 선택을 확인하면 유료로 전환할 수 있습니다.

9. 환불 정책

  • 유료(상품 구매) 전환 후 환불은 1주일 이내에만 가능합니다.
  • 상품을 1쪽이라도 유료로 전환해서 번역하면 구매했던 상품은 환불되지 않습니다.

10. 기타 (주의사항 등)

  • 유료 시, 언어 인식은 무료 언어 내에서 진행됩니다. 속도 문제로 일단 제거되었습니다.
  • 유료 시, PDF 파일은 대략 90일 정도 보관됩니다. 이는 더 짧아질 수 있으며, 유료 번역의 문제점을 파악하기 위한 용도로만 사용됩니다.
  • 유료 번역 시, 재번역을 하면 요금에서 차감됩니다. 이 점 유의하시기 바랍니다.
  • 유료를 위한 로그인 시 한 개의 디바이스에서만 가능합니다. 다른 디바이스에서 로그인하면 자동으로 로그아웃됩니다.

11. 업데이트 예정 사항

  • 유료 시, 글상자 기능은 아직 적용되지 않았습니다. 차후 업그레이드 시 진행할 예정입니다.
  • 번역 페이지 범위(시작, 종료) 기능을 추가할 예정입니다.
