CapCut 내보내기 가이드

AutoFlowCut에서 내보낸 프로젝트를 CapCut으로 가져오는 방법

🚀 빠른 시작 (4단계)

1

내보내기 모달에서 정보 입력

씬 목록 탭 > "📦 CapCut" 버튼 클릭 후 사용자명과 프로젝트 번호 입력

📦 CapCut 내보내기 ✕
✓

💡 터미널에서 whoami 입력

✓

💡 CapCut 폴더에서 다음 번호 확인

/Users/testUser/Movies/CapCut/User Data/Projects/com.lveditor.draft/0131
취소 📦 내보내기
2

ZIP 파일 압축 해제

다운로드된 프로젝트명_capcut.zip 파일을 더블클릭하여 압축 해제

3

CapCut 프로젝트 폴더에 복사

압축 해제된 폴더(예: 0131)를 아래 경로에 복사

4

CapCut 재시작

CapCut을 완전히 종료 후 다시 시작하면 프로젝트 목록에 표시됩니다

👤 시스템 사용자명 찾기

🍎 macOS
  1. 터미널 앱 실행 (Spotlight: Cmd + Space → "터미널")
  2. whoami 입력 후 Enter
  3. 표시되는 이름이 사용자명입니다
🪟 Windows
  1. 명령 프롬프트 실행 (Win + R → "cmd")
  2. whoami 입력 후 Enter
  3. 컴퓨터명\사용자명 형식으로 표시됨 (사용자명만 사용)

📁 CapCut 프로젝트 폴더 경로

🍎 macOS 
~/Movies/CapCut/User Data/Projects/com.lveditor.draft/

💡 Finder에서 Cmd + Shift + G 누르고 경로 붙여넣기

🪟 Windows

CapCut (기본)

C:\Users\[사용자명]\AppData\Local\CapCut\User Data\Projects\com.lveditor.draft\

CapCut Pro

C:\Users\[사용자명]\AppData\Local\CapCutPro\User Data\Projects\com.lveditor.draft\

Documents

C:\Users\[사용자명]\Documents\CapCut\Projects\

💡 파일 탐색기 주소창에 경로 직접 입력 또는 Win + E

📂 내보내기 폴더 구조

압축 해제 후 아래와 같은 폴더 구조가 나타납니다. 이 폴더 전체를 CapCut 프로젝트 경로에 복사하세요.

🔧 문제 해결

프로젝트가 CapCut에 나타나지 않아요

  • CapCut을 완전히 종료 후 다시 시작해보세요
  • 폴더 경로가 정확한지 확인하세요 (com.lveditor.draft 폴더 안에 복사)
  • 프로젝트 폴더 이름에 특수문자가 없는지 확인하세요

미디어 파일이 누락되었다고 나와요

  • media 폴더가 프로젝트 폴더 안에 있는지 확인하세요
  • ZIP 파일을 완전히 압축 해제했는지 확인하세요

어떤 경로를 선택해야 하나요?

  • AutoFlowCut 앱에서 프리셋 버튼(CapCut / CapCut Pro / Documents)을 바꿔가며 확인하세요
  • CapCut 프로젝트 폴더를 직접 열어서 기존 프로젝트가 있는 경로를 확인하세요
  • 일반적으로 CapCut이 기본 경로입니다

AppData 폴더가 보이지 않아요 (Windows)

  • 파일 탐색기 > 보기 > 숨긴 항목 체크
  • 또는 주소창에 %localappdata% 직접 입력
