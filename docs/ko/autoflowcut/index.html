💡 AutoFlowCut이란?

AutoFlowCut이란?

AutoFlowCut은 Google Flow AI(labs.google/fx)와 CapCut을 하나의 워크플로우로 연결하는 데스크톱 앱입니다. 단순한 이미지 생성기와 달리, AI 이미지 생성과 AI 비디오 생성(Text-to-Video, Image-to-Video)을 모두 지원하고, 모든 결과물을 바로 편집 가능한 CapCut 프로젝트로 내보냅니다.

🎬 AI 비디오 생성

AI 비디오 생성

텍스트나 이미지에서 비디오를 자동 생성합니다

📝

T2V — Text to Video

텍스트 프롬프트만으로 비디오를 생성합니다. 씬 설명을 입력하면 Google Flow AI가 움직이는 영상을 만들어줍니다. 레퍼런스 이미지 없이도 빠르게 비디오를 생성할 수 있어, 초기 컨셉 테스트나 빠른 프로토타이핑에 적합합니다.

↓입력: 텍스트 프롬프트
↑출력: AI 생성 비디오
★활용: 컨셉 영상, 빠른 프로토타입, 텍스트 기반 스토리보드
Text→Flow AI→Video
🖼️

I2V — Image to Video

생성된 AI 이미지를 움직이는 비디오로 변환합니다. 이미지의 캐릭터, 배경, 스타일을 유지하면서 자연스러운 모션을 추가합니다. 이미 완성된 이미지에 생동감을 더하고 싶을 때 사용합니다.

↓입력: AI 생성 이미지
↑출력: 애니메이션 비디오
★활용: 이미지 애니메이션, 캐릭터 모션, 배경 효과
Image→Flow AI→Video
🎯

스마트 미디어 선택

씬마다 이미지, T2V, I2V 중 최적의 결과물을 선택할 수 있습니다. 기본 우선순위: I2V → T2V → 이미지. CapCut 내보내기 시 선택한 미디어가 자동 반영됩니다.

I2V→T2V→Image

💬 FAQ

자주 묻는 질문

AutoFlowCut에 대해 알아야 할 모든 것

Q어떤 AI 모델을 사용하나요?
AutoFlowCut은 이미지와 비디오 생성 모두에 Google Flow AI(labs.google/fx)를 사용합니다. Flow API를 활용하여 레퍼런스 기반 스타일 매칭으로 일관된 고품질 비주얼을 생성합니다.
QWhisk2CapCut과 어떻게 다른가요?
Whisk2CapCut은 Google Whisk(이미지만)를 사용하지만, AutoFlowCut은 T2V(텍스트→비디오)와 I2V(이미지→비디오) 기능을 갖춘 Google Flow AI를 사용합니다. 씬별 미디어 선택과 Ken Burns 애니메이션도 지원합니다.
QGoogle 계정이 필요한가요?
네, Google Flow AI에 접근하려면 Google 계정이 필요합니다. 앱은 Google 세션을 통해 인증하므로 별도의 API 키는 필요 없습니다.
Q어떤 입력 파일 형식을 지원하나요?
TXT(줄 단위), CSV(컬럼 구조 데이터), SRT(타이밍 포함 자막 파일)에서 씬 프롬프트를 가져올 수 있습니다. 각 형식은 자동으로 씬으로 파싱됩니다.
QCapCut 데스크톱과 호환되나요?
네, 내보낸 프로젝트는 CapCut 데스크톱과 완전 호환됩니다. 타임라인, 미디어 파일, 자막, Ken Burns 애니메이션이 포함되어 바로 편집을 시작할 수 있습니다.
QAutoFlowCut은 무료인가요?
네, AutoFlowCut은 완전 무료 오픈소스입니다. 앱을 다운로드하거나 GitHub에서 소스를 빌드할 수 있습니다. Google Flow AI 사용도 실험 단계에서는 무료입니다.
