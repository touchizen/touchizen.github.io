💡 What is AutoFlowCut?

What is AutoFlowCut?

AutoFlowCut is a desktop app that connects Google Flow AI (labs.google/fx) and CapCut into one seamless workflow. Unlike simple image generators, AutoFlowCut supports both AI image generation and AI video generation (Text-to-Video, Image-to-Video), then exports everything as a ready-to-edit CapCut project.

💬 FAQ

Frequently Asked Questions

Everything you need to know about AutoFlowCut

QWhat AI model does AutoFlowCut use?
AutoFlowCut uses Google Flow AI (labs.google/fx) for both image and video generation. It leverages the Flow API to generate consistent, high-quality visuals with reference-based style matching.
QHow is it different from Whisk2CapCut?
Whisk2CapCut uses Google Whisk (image only), while AutoFlowCut uses Google Flow AI with T2V (Text-to-Video) and I2V (Image-to-Video) capabilities. AutoFlowCut also supports per-scene media selection and Ken Burns animations.
QDo I need a Google account?
Yes, you need a Google account to access Google Flow AI. The app uses your Google session for authentication — no separate API key is required.
QWhat file formats are supported for input?
You can import scene prompts from TXT (one per line), CSV (structured data with columns), and SRT (subtitle files with timing). Each format is automatically parsed into scenes.
QIs the CapCut project compatible with CapCut desktop?
Yes, the exported project is fully compatible with CapCut desktop. It includes timeline, media files, subtitles, and Ken Burns animations — ready to edit immediately.
QIs AutoFlowCut free?
AI image generation is free to try — Google Flow AI is currently free during its experimental phase. CapCut project export is a paid feature available via in-app purchase. The source code is open source on GitHub.
