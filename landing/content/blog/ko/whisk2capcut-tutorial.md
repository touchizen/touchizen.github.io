## Whisk2CapCut 소개: 생성부터 편집까지 원클릭

**Whisk2CapCut**은 Google Whisk의 AI 이미지 생성과 CapCut 영상 편집을 하나로 연결하는 데스크톱 앱입니다.

이미지를 하나씩 만들고, 하나씩 다운로드하고, 하나씩 타임라인에 배치하는 대신 — Whisk2CapCut은 이 모든 과정을 자동으로 처리합니다.

- 프롬프트 파일 하나로 **200장 이상의 AI 이미지를 일괄 생성**
- 생성된 이미지를 **CapCut 프로젝트로 즉시 내보내기** — 타임라인 배치, 자막, Ken Burns 애니메이션까지 자동 적용

더 이상 새벽 3시에 이미지를 하나씩 끌어다 놓을 필요가 없습니다.

## AI 영상 제작의 전체 워크플로우

AI 이미지 기반 영상을 처음부터 완성까지 만드는 과정입니다. Whisk2CapCut이 어디서 활약하는지 확인해 보세요.

### 1단계: 스크립트 작성

동화, 자기계발 쇼츠, 교육 콘텐츠 — 어떤 장르든 먼저 대본을 작성합니다. 이 대본이 전체 영상의 뼈대가 됩니다.

### 2단계: TTS로 음성과 자막 생성

텍스트를 음성으로 변환하는 TTS 도구를 사용해서 나레이션 오디오를 만듭니다. 대부분의 TTS 도구는 정확한 타임스탬프가 포함된 SRT 자막 파일도 함께 생성합니다. 이 타임스탬프가 장면을 나누는 기준이 됩니다.

### 3단계: SRT 타임스탬프로 장면 분할

SRT 파일의 각 자막 라인이 하나의 장면이 됩니다. Whisk2CapCut은 이 타임스탬프를 읽어서 몇 장의 이미지가 필요한지, 각 이미지가 얼마나 표시되어야 하는지를 자동으로 파악합니다.

### 4단계: 장면별 이미지 프롬프트 작성

각 장면에 맞는 이미지 프롬프트를 작성합니다. 직접 작성해도 되고, AI를 활용해서 대본에서 프롬프트를 자동 생성할 수도 있습니다. TXT, CSV, SRT 형식으로 저장하세요.

### 5단계: 이미지 일괄 생성 + CapCut 내보내기

**여기서 Whisk2CapCut이 등장합니다.** 프롬프트 파일을 불러오고 생성 버튼을 누르면:

- Google Whisk에 프롬프트를 순차적으로 전송
- 모든 이미지를 자동으로 일괄 생성
- 태그 매칭으로 캐릭터, 배경, 스타일의 시각적 일관성 유지
- 생성되는 즉시 로컬 드라이브에 자동 저장
- 타임라인 배치, Ken Burns 효과, 자막 트랙이 포함된 CapCut 프로젝트 파일로 내보내기

### 6단계: CapCut에서 최종 편집 후 발행

내보낸 프로젝트를 CapCut에서 열면, 이미지가 이미 타임라인에 배치되어 있고 자막도 싱크되어 있습니다. 나레이션 음성을 추가하고 마지막 수정을 한 뒤 바로 업로드하면 됩니다.

**밤샘 3일이 걸리던 작업이 1시간 이내로 줄어듭니다.**

## Whisk2CapCut 사용법 (단계별 가이드)

### 1. 프롬프트 파일 준비

이미지 프롬프트가 담긴 파일을 만듭니다. 세 가지 형식을 지원합니다:

- **TXT** — 한 줄에 프롬프트 하나. 가장 간단한 방식입니다.
- **CSV** — 프롬프트, 캐릭터 태그, 배경 태그, 스타일 태그를 열로 구분한 구조화된 형식입니다.
- **SRT** — 타임스탬프가 포함된 자막 형식. 각 장면의 정확한 재생 시간을 앱이 자동으로 인식합니다.

### 2. 태그 설정

Whisk2CapCut의 **자동 태그 매칭** 기능으로 시각적 일관성을 유지할 수 있습니다. 캐릭터 레퍼런스, 배경 스타일, 아트 스타일을 한 번만 설정하면 — 모든 생성 이미지에 동일하게 적용됩니다. 200장의 이미지가 하나의 영상에 어울리도록 만드는 핵심 기능입니다.

### 3. 이미지 일괄 생성

생성 버튼을 클릭하면 Whisk2CapCut이 프롬프트를 하나씩 Google Whisk에 전송합니다. 대기, 다운로드, 정리까지 모두 자동으로 처리됩니다. 생성되는 모든 이미지는 즉시 로컬 드라이브에 자동 저장됩니다.

### 4. CapCut으로 내보내기

생성이 완료되면 내보내기 버튼을 누르세요. Whisk2CapCut이 다음을 포함하는 `.capcut` 프로젝트 파일을 생성합니다:

- 모든 이미지가 타임라인에 순서대로 배치
- SRT 타임스탬프에 맞춘 재생 시간 설정
- **Ken Burns 애니메이션** (팬 & 줌) 자동 적용
- **자막**이 자막 트랙에 자동 배치

CapCut에서 열기만 하면 바로 편집을 시작할 수 있습니다.

## 주요 기능

- **AI 이미지 일괄 생성** — 프롬프트 파일 하나로 200장 이상의 이미지를 한 번에 생성합니다. 하나씩 다운로드할 필요가 없습니다.
- **시각적 일관성 유지** — 자동 태그 매칭으로 캐릭터, 배경, 아트 스타일이 전체 이미지에서 통일됩니다.
- **자동 저장** — 이미지가 생성되는 즉시 로컬에 저장됩니다. 작업물을 잃어버릴 걱정이 없습니다.
- **원클릭 CapCut 내보내기** — 이미지가 타임라인에 배치된 CapCut 프로젝트를 즉시 생성합니다.
- **Ken Burns 애니메이션** — 정적인 이미지에 자동으로 팬 & 줌 효과를 적용합니다. 키프레임을 직접 설정할 필요가 없습니다.
- **자동 자막 배치** — SRT 파일의 자막이 장면에 맞춰 자동으로 자막 트랙에 배치됩니다.
