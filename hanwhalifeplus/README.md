# LifePlus 

## How to install
1. 무엇보다 먼저,  node.js 를 설치하세요. 
2. 본 소스를 다운로드 하세요.
   
3. Next, To install packages for node.js
```
npm install
```

4. 스웨거 수정후
```
npm start
```

5. 확인하기 위해서
```
http-server ./docs
```
